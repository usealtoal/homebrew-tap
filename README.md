# Altoal Homebrew Tap

Homebrew formulae for [Altoal](https://github.com/usealtoal) projects.

## Install

```bash
brew install usealtoal/tap/edgelord
```

## Available Formulae

| Formula | Description |
|---|---|
| [edgelord](https://github.com/usealtoal/edgelord) | Prediction market arbitrage detection and execution CLI |

## Updating

```bash
brew update
brew upgrade edgelord
```
